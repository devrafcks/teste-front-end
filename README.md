Teste técnico para vaga de Desenvolvedor Front-End.

🔗 **Deploy:** https://econverse-rafael.vercel.app/

## Stack

- React + TypeScript
- Vite
- Sass
- TanStack Query
- Zod
- Axios

## Instalação e uso
```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview do build
npm run preview
```

## Funcionalidades

- Vitrine de produtos consumindo API externa
- Validação do schema da API com Zod
- Modal com detalhes do produto ao clicar
- Controle de quantidade no modal
- Tabs de categoria na vitrine
- Cache de 5 minutos com TanStack Query
- Layout responsivo
- Acessibilidade (ARIA, navegação por teclado, semântica HTML)
